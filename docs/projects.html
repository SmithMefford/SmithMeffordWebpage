Data Structures Final Project: Satellite Network.
A teamate and I coded a program manages a network of satellites with fast and secure access to individual satellites in the network. We wanted to build a system that lets an agency access a specific satellite directly and only from there, have access to its neighbors and so on through adjacency. This allows for sending messages between satellites, look at the network from a given perspective, and find the shortest path between satellites.
